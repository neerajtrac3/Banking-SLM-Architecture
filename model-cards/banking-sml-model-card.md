# BankingSLM – Model Card

## Model Overview
BankingSLM is a domain-specific Small Language Model designed for regulated financial institutions.

## Intended Use
- Payments
- Lending
- Compliance
- Core banking modernization

## Limitations
- Not a general-purpose LLM
- Requires domain-specific corpus
- Must be deployed with guardrails

## Ethical Considerations
- PII handling
- Regulatory compliance
- Explainability requirements

## Governance
Aligned with:
- UAGB (Unified AI Governance Blueprint)
- ERMI (Enterprise Resilience Maturity Index)
