# BankingSLM – Evaluation Framework

This framework defines how BankingSLM is evaluated for accuracy, safety, and compliance.

## 1. Domain Accuracy
- ISO 20022 semantic correctness
- Payment flow understanding
- Legacy code interpretation accuracy

## 2. Risk & Compliance Metrics
- Regulatory alignment
- PII leakage checks
- Explainability score

## 3. Safety Metrics
- Hallucination rate
- Determinism score
- Guardrail activation logs

## 4. Performance Metrics
- Latency
- Throughput
- Token efficiency
